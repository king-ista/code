Cocos2d-x is a engine PHÁT TRIỂN game đa nền tảng: iOS, Android, macOS, Windows, Linux. This is a game engine is a game engine was row up the member is used, phát triển trên 25 game chính thức, bao gồm nhiều tựa game nổi tiếng như: Piano Tiles - Don't Tap the White Tile, Hill Climbing Racing, FLow Free, Diamond Dash, Idle Heroes, AFK Arena… Cocos2d-x hỗ trợ 2 ngôn ngữ C ++ và Lua, nhưng chủ yếu là C ++. Tôi chọn cocos2d-x để làm hướng dẫn này cũng là một phần vì C ++, đây có thể nói là biến ngôn ngữ phổ biến nhất với các bạn mới bắt đầu lập trình. Tại sao chọn Cocos2d-x? - Dễ dàng cho người mới bắt đầu, API C ++ 11 hỗ trợ (auto, std :: function, lambda…). - Nền tảng đa nền - Với 1 mã nguồn có thể chạy trên cả máy tính để bàn và điện thoại di động. Có thể kiểm tra và gỡ lỗi trên máy tính và phát hành trên nền tảng di động. - API phong phú với đầy đủ các tính năng (sprites, hành động, hình ảnh động, hạt, chuyển đổi, bộ định thời, sự kiện (cảm ứng, bàn phím, gia tốc kế, chuột), âm thanh, tệp IO, sự bền bỉ, hình ảnh động bộ xương, 3D). - HOÀN TOÀN MIỄN PHÍ .
